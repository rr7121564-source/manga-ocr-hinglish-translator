# 📱 Phone mein APK Download Kaise Kare?

## ✅ Easiest Way - Direct Download

### Step 1: GitHub Actions ko Trigger Kare
```
Any commit kare repo mein
GitHub automatically APK build karega
```

### Step 2: Builds Check Kare
1. GitHub repo kholo: https://github.com/rr7121564-source/manga-ocr-hinglish-translator
2. **Actions** tab kholo
3. **Latest build** click karo
4. **Artifacts** section mein APK files dikhegi

### Step 3: APK Download Kare
- `app-debug.apk` (~60 MB) - Testing ke liye
- `app-release.apk` (~40 MB) - Better performance

## 📱 Phone mein Install Kaise Kare?

### Method 1: Direct Install (Easiest)

**Step 1: Phone mein Unknown Sources Enable Kare**
```
Settings 
  → Security / Privacy
  → Unknown Sources (ON)
```

**Step 2: APK Download Kare Phone mein**
- Chrome browser kholo
- APK download link se download karo
- Downloads folder mein save hoga

**Step 3: APK Install Kare**
```
File Manager kholo
Downloads → app-debug.apk
Tap karo → Install
Done! ✅
```

## ⚠️ Phone mein Install karke Problems?

### Problem 1: "App not installed"
- Phone storage full hai?
- Android version compatible nahi?
- Fix: Phone restart karo

### Problem 2: "Unknown Sources blocked"
```
Settings → Security → Unknown Sources (ON)
```

### Problem 3: "Cannot find download"
- Chrome Downloads folder check karo
- Ya different browser use karo
